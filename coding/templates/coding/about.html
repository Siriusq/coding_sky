About Coding Sky

Coding Sky is a MSc Project at the University of Glasgow. The author of the project is Kai Sun.

The application is based on a Coding Game developed by Unity, where players can learn coding by playing the game and viewing the reference answers given, write real Javascript code in the online code editor, or test their knowledge in the Quiz page.
