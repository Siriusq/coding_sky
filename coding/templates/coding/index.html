Coding Sky

Coding Sky is a web application developed for beginners to learn basic programming knowledge. The application is based on a Coding Game developed by Unity, where players can learn coding by playing the game and viewing the reference answers given, and can test their knowledge in the Quiz page.
For better user experience, please use a desktop or laptop to play the game.
If you want to use it offline, please download the corresponding version of the standalone game installation package.
